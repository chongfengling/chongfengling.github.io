# Chongfeng Ling's Personal Website and Blog

This repository hosts the source code for [Chongfeng Ling's personal website and professional blog](https://chongfengling.github.io/). Built with [Jekyll](https://jekyllrb.com/) and the elegant `al-folio` theme, this site serves as a platform to share insights, updates, and discussions related to my work as an AI engineer, with a focus on accelerating AI infrastructure and optimization.

## Getting Started

To view the website locally or contribute to its content:
1.  **Clone the repository:** `git clone https://github.com/chongfengling/chongfengling.github.io.git`
2.  **Navigate to the directory:** `cd chongfengling.github.io`
3.  **Install Jekyll:** Follow the instructions on the [Jekyll website](https://jekyllrb.com/docs/installation/).
4.  **Serve the site:** `bundle exec jekyll serve`
    The site will be available at `http://localhost:4000`.
